## 📥 설치 방법

### Windows
- 버전 확인
  - VS Code : 
  - Python : 
  - 라이브러리 : 
</br>

- 프로젝트 파일을 다운로드 받습니다.
```
git clone https://github.com/Jangsugyeong/election.git
```
- 프로젝트 경로에서 가상환경 설치 후 접속합니다. (Windows 10 기준)
```
virtualenv venv
source venv/Scripts/activate
```
- 라이브러리를 설치합니다.
```
pip install -r requirements.txt
```
